Privacy Policy

Last updated: 16 June 2026

This application is a personal productivity assistant ("the Service") that integrates with Google Workspace APIs (Google Calendar, Google Drive, Google Sheets, Google Docs, and Google Contacts) on behalf of the authorized user only.

Data Collected

The Service accesses Google Workspace data exclusively through authenticated OAuth 2.0 tokens. It reads and writes:

  • Calendar: Event metadata (titles, times, attendees)
  • Drive: File metadata and content for search, upload, and download operations
  • Sheets: Spreadsheet data for read and write operations
  • Docs: Document content for creation and editing
  • Contacts: Contact names and email addresses

How Data Is Used

All data is processed in-memory solely to fulfill user commands. No data is stored, logged, cached, or transmitted to any third party.

Data Storage

OAuth tokens are stored locally on the user's private server. No data is sent to external services beyond the Google APIs the user authorizes.

Data Sharing

No user data is ever shared, sold, or transferred to third parties. The Service has no analytics, no tracking, and no advertising.
